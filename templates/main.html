What is the deep Learning Image Classifier?

The deep learning Image classifier is a simple web application that allows user to use various pretrainded machine learning models that were trained in the imagenet dataset. You can click in the informative link in the model's tab to be directed to a pytorch implementation of that particular model. This can be used to quickly test a pretreined model in a particular image type for a better understanding of the precision and capability of each model
